# Linear & Multiple Linear Regression Notes

## Definition

Linear Regression is a machine learning algorithm used to predict a continuous value by finding the best relationship between input (X) and output (Y).

---

## Simple Linear Regression Equation

ŷ = mx + b

Where:

* x = input (independent variable)
* y = actual output
* ŷ = predicted output
* m = slope (rate of change)
* b = intercept (starting value)

---

## Multiple Linear Regression Equation

ŷ = m₁x₁ + m₂x₂ + ... + b

Where:

* x₁, x₂ = multiple input features
* m₁, m₂ = impact of each feature
* b = base value

---

## Slope (m)

* Represents how much output changes when input increases by 1
* Determines strength of relationship

### Cases:

* m > 0 → positive relationship
* m < 0 → negative relationship
* m = 0 → no relationship

---

## Intercept (b)

* Value of output when all inputs = 0
* Starting point of the model

---

## Goal of Linear Regression

To find the best values of m (or m₁, m₂, …) and b such that predictions are as close as possible to actual values.

---

## Prediction

ŷ = mx + b
(or for multiple inputs: ŷ = m₁x₁ + m₂x₂ + b)

---

## Error (Mean Squared Error - MSE)

MSE = (1/n) Σ (y - ŷ)²

* Measures difference between actual and predicted values
* Lower MSE = better model

---

## Steps in Linear Regression

1. Load dataset
2. Separate input (X) and output (Y)
3. Split data into training and testing
4. Train model
5. Predict values
6. Evaluate using MSE
7. Use model for new predictions

---

## Train-Test Split

* Training data → used to train model
* Testing data → used to evaluate model

Purpose:

* Check performance on unseen data
* Avoid overfitting

---

## Feature Scaling (Important - Day 3)

* Different features can have different ranges
* Example:

  * size → 1000–2000
  * bedrooms → 1–5

Problem:

* Model becomes unbalanced

Solution:

* Use StandardScaler to normalize data

Important Rule:

* If you scale training data → always scale new input before prediction

---

## Coefficients (m values)

* Represent importance of each feature
* Higher coefficient = stronger impact
* Cannot compare directly without scaling

---

## Overfitting vs Underfitting

* Overfitting:
  Model memorizes training data and performs poorly on new data

* Underfitting:
  Model is too simple and cannot learn pattern

---

## Real-Life Examples

* House price prediction
* Salary prediction
* Sales forecasting

---

## Important Points

* Assumes linear relationship
* Sensitive to outliers
* Easy to understand and implement
* Works well for simple problems
