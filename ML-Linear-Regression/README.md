# 📊 Linear Regression Project

## 🚀 Overview

This project predicts house prices using Linear Regression.

## Dataset

* Input: House size (sq.ft)
* Output: Price (in lakhs)

## Tech Stack

* Python
* Pandas
* Scikit-learn

## Results

* MSE: ~2.3
* Prediction for 1900 sq.ft: ~77 lakhs
